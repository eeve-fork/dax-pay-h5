# DaxPay移动H5端

## 简绍
vite5 + vue3.4 + vben admin2 + antd vue4 搭建

## 环境依赖

| 环境   | 版本        | 备注                       |
|------|-----------|--------------------------|
| Node | >=20.9.0	 | 低版本会无法进行构建               |
| pnpm | >=9.0.2   | 低版本会导致 pnpm-lock.yaml 失效 |

## 启动命令

```shell
# 安装项目依赖
pnpm install
# 启动项目
pnpm run dev
# 构建打包
pnpm build
```
